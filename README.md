# Sistema de Farmacia (Java)

## Versión
**1.0**

## Descripción
Sistema de gestión de farmacia desarrollado en Java. Permite registrar clientes, proveedores, productos (medicamentos, suplementos, cuidado personal), control de inventario, alertas de stock bajo, recetas médicas y ventas.

## Alcance

### ✅ Qué hace
- Registrar clientes y proveedores
- Gestionar inventario por categorías (Medicamento, Suplemento, Cuidado Personal)
- Control de stock y alertas de stock mínimo
- Registrar medicamentos con principio activo, dosis, receta y fecha vencimiento
- Control de medicamentos vencidos
- Registrar recetas médicas con diagnóstico
- Realizar ventas con validación de receta para medicamentos controlados
- Historial de ventas por cliente

### ❌ Qué NO hace
- No tiene interfaz gráfica (solo consola)
- No persiste datos en base de datos real

## Tipos de Producto

| Tipo | Características |
|------|-----------------|
| Medicamento | Principio activo, dosis, requiere receta, fecha vencimiento |
| Suplemento | Tipo, presentación, sabor |
| Cuidado Personal | Tipo, uso, hipoalergénico |

## Categorías

| Categoría | Descripción |
|-----------|-------------|

| SUPLEMENTO | Vitaminas y suplementos alimenticios |
| CUIDADO_PERSONAL | Productos de higiene y cuidado personal |
